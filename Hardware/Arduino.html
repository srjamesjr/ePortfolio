This is my final Embedded controller course project: A portable Emulator. It’s a raspberry pi, running an emulator, to a 480p LCD screen, with a breadboard of buttons, duct taped to a hollowed ps2 controller, running on a battery bank.

I didn’t have a lot of time or resources to put this project together. I am using Mednafen emulator, and a background process to read the button presses and simulate keyboard presses. You could also connect it up to a monitor with HDMI and the display would auto switch. I only had half a week to do this project so I couldn’t 3d print or perf-board any components, which makes it look a lot less cool than it is
